# Emotion Detection from Face using AI

## Project Overview
This project is an AI-based facial emotion detection system developed using Python and image processing techniques. The system captures facial expressions through a camera and identifies the emotion displayed by the person.

## Features
- Real-time face detection using a camera
- Detects facial expressions
- Identifies different human emotions
- Displays the detected emotion on the camera screen
- Simple and easy-to-use Python implementation

## Technologies Used
- Python
- OpenCV
- Artificial Intelligence
- Machine Learning
- Digital Image Processing

## How It Works
1. The camera captures a live video stream.
2. The system detects the face present in each video frame.
3. The detected facial region is processed.
4. The emotion detection model analyzes the facial expression.
5. The predicted emotion is displayed on the video screen.
